VESPY

VESPY es una aplicación desarrollada en Python para la visualización y procesamiento de datos de Sondeo Eléctrico Vertical (VES). Está diseñada para ingenieros y científicos interesados en el análisis de resistividad del subsuelo.

Principales funcionalidades:

  • Carga y procesamiento de datos desde archivos Excel, CSV u ODS.
  • Inversión de resistividad para obtener modelos del subsuelo.
  • Visualización gráfica de resistividad y análisis estadísticos.
  • Gestión de modelos invertidos para análisis y visualización.
  • Interfaz gráfica interactiva construida con PyQt5.

Tecnologías: Python, PyQt5

HVSRLEARN

HVSRLEARN es una aplicación en Python diseñada para el cálculo y análisis de la Relación Espectral Horizontal a Vertical (HVSR) a partir de datos sísmicos en tres componentes (Z, N, E). La aplicación cuenta con una interfaz gráfica construida con Tkinter, proporcionando una plataforma intuitiva e interactiva para analizar y visualizar datos sísmicos.

Principales funcionalidades:

  • Interfaz fácil de usar: Controles organizados para facilitar la interacción.
  • Visualización en tiempo real: Espectro HVSR y parámetros relevantes ajustables en tiempo real.
  • Proceso de análisis personalizable: Configuración de parámetros como sobremuestreo, ancho de ventana, solapamiento y método HVSR.
  • Capacidad para guardar resultados: Gráficos y datos relevantes para referencia futura.
  • Compatibilidad con varios métodos HVSR: Flexibilidad en el enfoque de análisis.
  • Monitoreo mediante consola: Información detallada sobre procesos y resultados.
  • Gestión de archivos: Búsqueda y carga de datos sísmicos.

Tecnologías: Python, Tkinter
