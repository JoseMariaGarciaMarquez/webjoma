GPRPy Extra

Premium Python GPR Patreon

Paquete de extensiones premium para GPRPy, exclusivo para suscriptores de Patreon. Convierte GPRPy en una plataforma profesional con capacidades de software comercial.

Módulos Avanzados:

  • 🔧 Extensions: Métodos premium agregados automáticamente
  • 🎛️ Advanced Filters: Wiener, morfológicos, adaptativos
  • 📡 Migration: Kirchhoff, RTM, F-K algorithms
  • 📊 Velocity Analysis: Semblanza, CMP, tomografía
  • 🤖 Interpretation: Pickeo automático de horizontes
  • 📤 Export Tools: LAS, SEG-Y, Geosoft, Petrel, MATLAB

💡 Ejemplo de Uso:

import gprpy.gprpy as gp
import gprpy_extra

mygpr = gp.gprpyProfile("archivo.DT1")
mygpr.kirchhoff_migration_advanced(velocity=0.08)
velocities, semblance = mygpr.semblance_analysis((0.05, 0.15))
horizons = mygpr.automatic_horizon_picking(num_horizons=3)
mygpr.export_las("output.las")
